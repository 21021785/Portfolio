Through this website, people can read articles, see tour packages, find out about Singapore, find out about cruises and contact the company. We were tasked to make everything ourselves, including the logo. We were also told not to use any frameworks and were limited to HTML and CSS only.

Exercise Tracker

This website can help keep track of someone's exercise. It has running, cycling and weightlifting as the 3 stages the user must complete. The user must complete each activity as well as its requirements before moving to the next stage.
The user will be asked to input their statistics over the week and a chart will appear, visualising the user's activity and will either allow them to proceed to the next activity or inform them that they have not met the requirement.
It was made as an end-module project in one of my modules and was made in around a week.

Technologies used: HTML, CSS, JavaScript, Bootstrap, Chart.js, Anime.js, Fullpage.js

Hotel Review and Booking

This website can allow a user to view, book and rate hotels around Singapore. It also checks who is being logged in, which affects their access. For example, admins have access to deleteing anybody's review and users can only post, edit and delete their own review.
Using cookies, the website knows who is being logged in, which affects functionality, as stated above. Login details can also be saved for easier log in when a user logs out.
There have been cybersecurity features implented such as hashing and prevention of SQL injection.
The reason that there is no Github website for this is because it contains data from a local database set up in mysql and that Github cannot host PHP websites.
It was made as an end-module project in one of my modules and was made in 1 day.

Note: This is purely front-end work, there is no connection to current data and data is not updated.

Technologies used: HTML, CSS, PHP, JavaScript, Bootstrap
